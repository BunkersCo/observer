Admin Menu

  Device Settings: การตั้งค่าของผู้เล่น
  Device Manager: ผู้จัดการผู้เล่น
  Device Monitoring: การตรวจสอบผู้เล่น

Device Manager

  Heading: ผู้จัดการผู้เล่น
  New Device Button: ผู้เล่นใหม่
  ID: ID
  Name: ชื่อ
  Description: ลักษณะ
  Status: สถานะ

  Version: รุ่น
  Last IP: IP ล่าสุด
  All: ทั้งหมด
  Schedule: ตารางเวลา
  Emergency: ลำดับความสำคัญ
  Media: สื่อ
  Playlog: Playlog

  not available: ไม่สามารถใช้ได้
  never: ไม่เคย

  Form Heading: การตั้งค่า
  Settings Name: ชื่อ
  Settings Description: ลักษณะ
  Settings Parent Device: ผู้เล่นผู้ปกครอง
  Schedule Check: ตารางเวลาการใช้งานของผู้ปกครอง
  Dynamic Check: ผู้ปกครองเลือกใช้แบบไดนามิก
  ID Check: ผู้ปกครองใช้รหัสสถานี
  Playlist Check: ใช้ผู้ปกครองเริ่มต้นรายการ
  Emergency Check: ใช้ออกอากาศลำดับความสำคัญของผู้ปกครอง

  IP Address: ที่อยู่ IP
  Password: รหัสผ่าน
  Enter New Password: (ใส่รหัสผ่านใหม่เพื่อตั้งค่า)
  Timezone: เขตเวลา
  Default Playlist: เริ่มต้นเพลย์ลิส
  Playlist Drag Zone: (ลากรายการที่จะเล่นที่นี่)
  Media Drag Zone: (ลากสื่อที่นี่)
  Station IDs: รหัสสถานี
  Station ID Image Duration: สถานี ID ภาพระยะเวลา
  seconds: วินาที
  Media Types: ประเภทสื่อ
  Dynamic Selections Only: มีผลต่อการเลือกแบบไดนามิกเท่านั้น
  Audio: เสียง
  Images: รูปภาพ
  Video: วีดีโอ
  Line-In: สายใน

  None: ไม่

  Device List Must Load: รายชื่อผู้เล่นจะต้องโหลดก่อนที่จะสร้างผู้เล่นใหม่
  Saved Message: ผู้เล่นที่บันทึกไว้: (ID #%1)

Device Monitoring

  Options: ตัวเลือก
  Device: ผู้เล่น
  Start: เริ่มต้น
  End: ท้าย
  Sort: ประเภท

  Media ID: ID สื่อ
  Artist: ศิลปิน
  Title: ชื่อเรื่อง
  Date/Time: วันที่ / เวลา
  Context: สิ่งแวดล้อม
  Notes: หมายเหตุ

  Ascending: จากน้อยไปมาก
  Descending: มากไปหาน้อย

  Filters: ตัวกรอง

  Exactly: คือว่า
  Not: ไม่ใช่
  Contains: ประกอบด้วย
  Not Contain: ไม่ได้มี

  Add Filter Button: เพิ่มตัวกรอง
  Load Data Button: โหลดข้อมูล

  Results: ผล
  No Results Message: ไม่มีผลลัพธ์การค้นหา
